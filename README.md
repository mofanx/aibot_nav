# AI导航项目 (aibot_nav)

## 项目介绍

这是一个AI导航项目，使用TypeScript和Vite构建的现代化前端应用。该项目旨在提供一个用户友好的AI工具导航平台，帮助用户快速找到并使用各种AI工具和服务。

## 技术栈

- **前端框架**：TypeScript + Vite
- **开发工具**：Bun
- **代码规范**：ESLint, Prettier, Biome
- **构建工具**：Vite

## 功能特点

- 现代化UI界面
- 快速导航到各种AI工具
- 响应式设计，适配各种设备
- 高性能加载体验

## 开发指南

### 环境要求

- Node.js (推荐最新LTS版本)
- Bun

### 安装依赖

```bash
bun install
```

### 开发模式

```bash
bun run dev
```

### 构建项目

```bash
bun run build
```

### 预览构建结果

```bash
bun run preview
```

### 代码检查

```bash
bun run lint
```

## 部署

项目配置了Netlify部署文件，可以轻松部署到Netlify平台。
